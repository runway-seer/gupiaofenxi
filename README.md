# AI 股票分析系统
基于 Java + Vue + AI 大模型实现的智能股票分析网页

## ✨ 项目功能
- 股票实时数据展示
- K线图 / 走势图可视化
- AI 智能分析股票趋势、风险、投资建议
- 前后端分离架构
- 简洁美观的网页端

## 🛠 技术栈
- 前端：Vue3 / JavaScript / Element Plus
- 后端：Java / SpringBoot
- AI 能力：对接大模型进行股票分析
- 数据：股票行情接口

## 📄 介绍
这是一个完整的 AI 股票分析工具，用户可以在网页上查看股票数据，并获得由 AI 生成的专业分析报告，适合学习、演示、二次开发。
